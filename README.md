# Monorepo

Монорепозиторий на базе pnpm и Turborepo.

## Структура

- `apps/web` - Основное веб-приложение (Vite + React + TypeScript)
- `apps/admin` - Админ-панель (реюз web через маршруты)
- `packages/ui` - Общие UI компоненты (shadcn/ui)

## Технологии

- **pnpm** - менеджер пакетов
- **Turborepo** - система сборки для монорепозиториев
- **Vite** - сборщик
- **React** - UI библиотека
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **PWA** - Progressive Web App поддержка

## Установка

```bash
pnpm install
```

## Разработка

Запуск всех приложений в режиме разработки:

```bash
pnpm dev
```

Запуск конкретного приложения:

```bash
cd apps/web
pnpm dev
```

или

```bash
cd apps/admin
pnpm dev
```

## Сборка

```bash
pnpm build
```

## Темы

Приложение поддерживает две темы:
- **Light** - светлая тема (по умолчанию)
- **Dark Granite** - темная гранитная тема

Темы настраиваются через CSS-переменные в `index.css`.

## PWA

Оба приложения настроены как Progressive Web Apps с поддержкой:
- Автоматического обновления
- Офлайн режима
- Установки на устройство

## Зависимости

Все основные зависимости установлены:
- `@tanstack/react-query` - управление состоянием сервера
- `zustand` - управление состоянием клиента
- `zod` - валидация схем
- `react-hook-form` - работа с формами
- `axios` - HTTP клиент
- `framer-motion` - анимации
- `lucide-react` - иконки
- `clsx` - утилиты для классов
