# Cloudflare Tunnel - Опциональная функция

## Что это?

Cloudflare Tunnel — это опциональная функция для создания защищенного туннеля между вашим сервером и Cloudflare. Она **НЕ обязательна** для работы приложения.

## Когда нужен?

- Если вы хотите использовать Cloudflare для проксирования трафика
- Если нужен дополнительный уровень защиты
- Если хотите скрыть реальный IP сервера

## Когда НЕ нужен?

- Если у вас уже есть домен с SSL (например, `optmramor.ru`)
- Если вы используете обычный Nginx/другой веб-сервер
- Если не нужны дополнительные функции Cloudflare

## Настройка

В `.env.production` просто оставьте поле пустым:

```env
CLOUDFLARE_TUNNEL_URL=
```

Или вообще не указывайте это поле — приложение будет работать нормально.
